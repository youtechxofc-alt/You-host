API Reference

Intégrez YABZ dans vos bots, scripts & apps.

POST
/api/upload

Upload un fichier unique. Retourne l'URL et les métadonnées.

curl -X POST /api/upload \ -F "file=@image.png" # Réponse { "url": "https://yabz.io/f/uuid", "id": "uuid", "size": 204800 }
POST
/api/upload/multi

Upload plusieurs fichiers (max 10). Retourne un tableau d'URLs.

curl -X POST /api/upload/multi \ -F "files=@img1.jpg" \ -F "files=@video.mp4" # Réponse { "urls": ["...", "..."], "count": 2 }
GET
/api/file/:id

Obtenir les métadonnées d'un fichier par son ID.

curl /api/file/uuid # Réponse { "id": "uuid", "url": "https://...", "type": "image/png", "size": 204800, "created": "2026-05-11" }
